1. O que é o HTML?

    • a) Uma linguagem de marcação responsável por definir a estrutura de um documento
    • b) Uma linguagem de programação que funciona por meio de uma série de instruções, símbolos, palavras-chave, regras semânticas e sintáticas.
    • c) Uma classe que define um tipo de dado em uma linguagem orientada a objeto
    • d) Uma variável que é um objeto capaz de reter e representar um valor ou expressão.

  2. Qual a função da tag < !DOCTYPE html >?

    • a) Indicar para o browser o tipo e versão do documento que está sendo aberto.
    • b) Conter todo os elemento de código
    • c) Incluir uma imagem ao texto
    • d) Representar uma lista de itens sem ordem rígida

  3. Qual das tags são estruturais, ou seja quais das tags têm grande importância na questão semântica na página?

    • a) < header > – Todo conteúdo que estiver dentro dela faz parte de um cabeçalho, podendo ser utilizado dentro de outras sessões.
    • b) < b > - Transforma o conteúdo em negrito
    • c) < i > - Transforma o conteúdo em itálico
    • d) < p > - Principal tag de texto, compõe um parágrafo

  4. Qual das tags não necessita de fechamento?

    • a) < img >
    • b) < body >
    • c) < ul >
    • d) < li >

  5. As tags de título possuem valor semântico, variando entre seis níveis hierárquicos. Qual a tag de maior nível?

    • a) < h1 >
    • b) < p >
    • c) < h6 >
    • d) < h2 >
